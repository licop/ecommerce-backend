## 电商项目后端代码

使用 `empress` + `mongodb` 开发的电商项目后端代码

使用前将 db 文件夹中的 json 数据导入`mongodb`数据库即可

导入数据命令参考如下

```js
mongoimport --db ecommerce --collection categories --file /Users/licop/Documents/ecommerce-backend/db/categories.json
```
